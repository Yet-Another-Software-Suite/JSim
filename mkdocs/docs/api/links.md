 # API Guide — Cross-links

This collection of pages provides human-readable guidance for the codebase API. Instead of linking to generated API artifacts, each page explains where to find implementation and usage examples in the repository and how to apply the API in C++, Python, and Java.

Key locations
- Core implementation: `core/src`, `core/include`
- CAD import: `cad-import`
- Python bindings and examples: `python/` and `examples/python`
- Java bindings and examples: `jsim/java` and `jsim/examples`

View the generated API under `/api/` in the site output.
