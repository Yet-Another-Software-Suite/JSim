# Functions — Usage Guide

Common function categories
- Creation/factory functions: create worlds, parsers, and model builders.
- Math helpers: vector/matrix/quaternion operations are provided in the math modules.
- IO functions: loaders for URDF/STL, exporters, and material parsers.

Examples
- See `examples/python/simple_world_demo.py` to observe a complete flow: instantiate, configure, step, record.
