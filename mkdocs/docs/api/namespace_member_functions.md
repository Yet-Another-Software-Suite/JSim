# Namespace Functions — Guide

Common functions
- Utility functions live next to the math and geometry helpers (see `cad-import/geometry_utils.py`).
- Factory and parser functions for URDF/STL live in `cad-import/importer.py` and related modules.

How to learn them
- Inspect example usage in `examples/` and in `cad-import/examples`, then open the implementation files to review parameter lists and return types.
